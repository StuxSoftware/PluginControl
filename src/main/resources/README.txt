             > A Server-Platform independent Plugin Controller! <

Description
-----------
This  plugin  provides  a way to manager your plugins across multiple platforms.
This  allows you to install, remove and control your plugins in the same way! In
Bukkit and Canary-Mod.

Features
--------
   + Load and Unload of plugins.
   + Disable and enable of plugins.
   + A fine permission control.
   + Works in the same way in all servers.

Installation
------------
Just extract the jar-file out of the zip file into the plugin directory of your
server.

Notes
-----
This is the successor of BukkitInstall. Please do not install both plugins on
the same server.

Documentation
-------------
A documentation is stored in the DOCUMENTATION.txt-File in the zip-File.

Licenses
--------
All license information is stores in the LICENSES.txt-File shipped within the
zip-Folder.
